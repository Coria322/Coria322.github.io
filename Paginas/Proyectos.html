Proyectos

Inventario POI

Este sistema fue diseñado en Java, utilizando Apache POI para facilitar la manipulación de un set de datos en excel con más de 100 mil entradas. Por medio de linea de comando permite registrar, consultar, modificar y eliminar registros del inventario. Con esta herramienta fue posible facilitar la gestión del inventario del cliente, sin la necesidad de migrar a un sistema más complejo y costoso

Sistema Clickwear

Sistema de punto de venta sin contacto para la tienda de ropa clickwear. Este proyecto fue realizado en Java, utilizando dependencias como Javax Mail y pdfbox. Así como una base de datos SQL

DuckHunt Game

Un juego basado en el duckhunt original, diseñado en Java, implementando herramientas de programación paralela y Graphics 2D de Java. Permitiendo una experiencia dinámica y divertida

Planetario IA

Sistema de planetario interactivo, realizado en Python. Utilizando las API de gemini y SpeechRecognition de google. Fue diseñado para Estudiar la interacción Humano - Computadora; Así como los métodos de interacción con los sistemas informáticos. Entre sus carácteristicas más importantes se encuentran: Interacción por voz, respuesta con IA de preguntas relacionadas al planeta, reconocimiento del planeta mostrado.
